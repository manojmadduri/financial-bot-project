Advanced Analysis Feature

Our advanced analysis command !analyze SYMBOL empowers you with technical insights by:

  • Retrieving historical daily stock data via the Alpha Vantage API.
  • Calculating key technical indicators such as the 20-day and 50-day Simple Moving Averages (SMA).
  • Generating an informative chart that overlays closing prices with the moving averages.

How It Works

When you enter the command (for example, !analyze TSLA) in your Discord channel, our bot:

  1. Fetches the latest historical data for the specified stock symbol.
  2. Processes the data using pandas to sort dates and compute the SMAs.
  3. Uses matplotlib to create a chart that displays:
    • The daily closing price
    • The 20-day moving average
    • The 50-day moving average
  4. Sends the chart image directly to the Discord channel so you can quickly assess market trends.
